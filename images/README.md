## Image Specifications:
- **Format:** JPG (optimized for web)
- **Minimum Resolution:** 1200px width
- **File Size:** Optimized to under 500KB each
- **Aspect Ratio:** 4:3 or 16:9 for most images
- **Quality:** Professional, well-lit, showing actual drywall work

## Image Optimization:
Before uploading, optimize images using:
- TinyPNG.com
- Squoosh.app
- ImageOptim (Mac)
- Or any other image compression tool
